# Bank Loan Analysis Dashboard (Excel)

## Project Overview
This project analyzes customer loan data using Microsoft Excel. It provides an interactive dashboard to monitor loan acceptance, customer demographics, income, education, online banking usage, and credit card adoption.

## Features
- Interactive KPI Cards
- Pivot Tables
- Pivot Charts
- Dynamic Slicers
- Loan Acceptance Analysis
- Average Income by Education
- Average Age by Loan Status
- Credit Card Customer Analysis
- Online Banking Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
